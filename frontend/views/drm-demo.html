How it works

When you click "Attempt Authorized Playback", the Shaka Player networking engine is configured with a requestFilter. This filter intercepts the license request and appends your SIWE signature and the NFT Token ID to the HTTP headers.

Our DRM Proxy server receives the request, recovers your address from the signature, and verifies on-chain that you are the current owner of the NFT. Only then does it return the decryption keys (or in this PoC, a successful mock response).
